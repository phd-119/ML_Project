📊 Customer Retail ML Dashboard
🚀 About
   A Machine Learning project that analyzes customer retail data and compares multiple models to predict outcomes and            generate insights.
🧠 Models Used
    Logistic Regression
    Decision Tree
    K-Nearest Neighbors (KNN)
📦 Requirements
    Python 3.8+
     numpy
    pandas
     matplotlib
     seaborn
    scikit-learn
     Install
      Bash
    pip install -r requirements.txt
📁 Project Structure
    data/ → Dataset
     notebooks/ → EDA & training
     models/ → Saved models
    src/ → Python scripts
    images/ → Screenshots
▶️ Run Project
   Bash
  python app.py

  Best model: Decision Tree (example)
